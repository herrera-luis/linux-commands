## VirtualBox

Instalar `VirtualBox` [aquí](https://www.virtualbox.org/wiki/Linux_Downloads).


## Instalar Vagrant

Descargar desde [aquí](https://www.vagrantup.com/downloads.html).

Instalar con `gdebi`.

## Aprovisionar

Crear Vagrantfile:
```
$ vagrant init hashicorp/bionic64
```

Correr:
```
$ vagrant up
```

Eliminar:
```
$ vagrant destroy
```

Entrar por ssh:
```
$ vagrant ssh
```
